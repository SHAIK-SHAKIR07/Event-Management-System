## вҡЎ Getting Started

### Clone the Repository

```bash

```

### Install Dependencies

```bash
cd server
npm install

cd ../client
npm install
```

### Start the Application

```bash
# Backend
cd server
npm run dev

# Frontend
cd client
npm start
```
